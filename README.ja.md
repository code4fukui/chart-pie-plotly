# chart-pie-plotly

Plotly.jsを使用して、円グラフを表示するWebコンポーネントです。CSVデータや手動で指定したデータをもとに、円グラフを描画することができます。

## デモ

https://code4fukui.github.io/chart-pie-plotly/

## 機能

- CSVデータから自動で円グラフを描画
- 手動でデータを指定して円グラフを描画
- オプションでグラフのサイズや凡例の表示を指定可能

## 必要環境

- Plotly.js

## 使い方

HTMLファイルでchart-pieタグを使用することで、簡単に円グラフを表示できます。CSVデータを直接記述するか、外部のCSVファイルを指定することができます。

```html
<script type="module" src="./chart-pie.js"></script>
```
